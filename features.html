Hangman Create Task Practice Project -- AP Computer Science Principles

The objective of this project was to create a working hangman game. My partner and I created a game in python. Some requirements that needed to be met were that we needed a loop that has sequencing, selection, and iteration. Another requirement was to have a function that used parameters that could have different arguments. The code itself has fulfilled the requirements of the AP create task. On top of this the code also features arrow handling, a superhero theme, and more! My partner and I actually used pair programming to work on the code instead of coding separately and this made our coding faster because we wouldnot need to worry about combining our code and deal with that every time we finished one task. Two specific skills that were used by me were coding, and peer-review.
